## 引言

> **声明**：以下介绍的，就是“安全”清理 C 盘的所有方法。这些操作对电脑系统和个人数据没有任何影响，请放心食用。

很多朋友用电脑一段时间后，都会面临 C 盘飘红的窘境。不仅影响运行速度，看着也让人焦虑。今天这篇教程，就带你一步步把 C 盘的垃圾文件清理干净，并养成好习惯。

---

## 核心清理方案

### 方法一：清理系统隐藏临时文件夹

系统和软件在运行过程中会产生大量的临时缓存，这些文件通常被隐藏得很深，但我们可以通过快捷指令直接触达。

1. 使用快捷键 <kbd>Win</kbd> + <kbd>R</kbd> 调出“运行”窗口。
2. 在输入框中输入 `%temp%`，然后点击“确定”。
3. 在弹出的文件夹中，按下 <kbd>Ctrl</kbd> + <kbd>A</kbd> 全选所有文件。
4. 使用快捷键 <kbd>Shift</kbd> + <kbd>Delete</kbd> 永久删除这些文件。

_注：如果有弹出提示“操作无法完成，因为文件已在另一程序中打开”，勾选“为所有当前项目执行此操作”后点击“跳过”即可。_

### 方法二：深度使用“磁盘清理”工具

Windows 自带的磁盘清理工具非常强大，但很多时候我们没有挖掘出它的全部潜力。

1. 打开“此电脑”，右键点击 C 盘，选择“属性”，然后点击**磁盘清理**。（Windows 11更新为详细信息）
2. 在弹出的窗口下方，点击带有盾牌图标的 **清理系统文件** 按钮（这一步能帮你扫出很多陈年的 Windows 更新残留）。
3. 扫描完成后，在“要删除的文件”列表中，你可以单击每一项查看下方框内的具体描述。
4. **如无特殊需求，全部勾选清理即可**。

### 方法三：通过系统“设置”清理存储

Windows 10/11 的现代化设置面板也提供了直观的清理入口。

1. 按下 <kbd>Win</kbd> + <kbd>I</kbd> 打开 Windows 设置，点击 **系统**。
2. 在左侧菜单栏选择 **存储**。
3. 在右侧磁盘使用情况中，点击 **临时文件**。
4. 系统会自动扫描可回收的空间，勾选需要删除的项目，点击 **删除文件** 即可。

## 进阶：从源头杜绝空间占用

### 1. 迁移“存储大户”的数据（以微信为例）

像微信这类通讯软件，长期使用会接收大量的图片、视频和文件，是名副其实的“C 盘杀手”。我们需要把它的数据存储位置改到其他盘。

1. 打开微信电脑版，进入左下角的“设置”。
2. 选择“文件管理”。
3. 找到“文件存放位置”，点击 **更改**，将其移动到 D 盘或任何非系统盘的目录下。

### 2. 养成良好的软件安装习惯

这是最根本的解决方法。在安装任何新软件时，千万不要习惯性地一路狂点“下一步”或者“安装”。

请务必注意查看 **安装路径**，手动将其中的首字母 `C` 更改为 `D`（例如将 `C:\Program Files\Tencent\Weixin` 改为 `D:\Program Files\Tencent\Weixin`）。

---

> 只要定期执行以上清理步骤，并保持良好的安装与文件管理习惯，你的 C 盘就能长久保持健康状态。
